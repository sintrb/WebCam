# WebCam Release

## 下载

- APK: `WebCam-v1.0-inruan-release.apk`

## 更新内容

- Android 本地摄像头预览
- HTTP 抓拍接口 `/snapshot.jpg`
- 支持尺寸、闪光灯、延时、旋转参数
- 支持自动对焦
- 支持时间水印位置配置
- 支持网页端最近抓拍记录与图片预览
- 支持远程设置

## 安装说明

1. 下载 APK
2. 安装到 Android 手机
3. 首次打开时授予摄像头权限
4. 在同一网络下访问：

```text
http://手机IP:8888/
```
